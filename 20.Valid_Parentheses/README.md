# 20. Valid Parentheses
#### tags:
- stack

## 題目：
給定一個只包含字符 '(', ')', '{', '}', '[', ']' 的字串 s，判斷這個字串是否是有效的。

一個有效的字串需要滿足以下條件：

開括號必須由相同類型的閉括號關閉。
開括號必須以正確的順序關閉。
每個閉括號都有對應的相同類型的開括號。

## 可使用的方法：
- stack (LIFO)

## 解題思路： 
建立對應括號映射對應
新增空 stack 用來儲存左括號
如果是右括號, stack 為空或當前不匹配則 false
如果最後 stack 未消除 則有未消除左括號
